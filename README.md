 Sabor Express

Projeto simples feito pra praticar Programação Orientada a Objetos (POO) em Python.  
Desenvolvido durante meus estudos na Alura, aplicando classes, métodos, construtores, listas e métodos especiais.

---
 O que o projeto faz
 
- Cria e gerencia restaurantes
- Registra avaliações
- Mostra uma listagem formatada no terminal
- Usa boas práticas de POO, com `@property`, `__init__`, `__str__` e métodos de classe

---
Conceitos praticados

Classes e Objetos
Encapsulamento (atributos protegidos)
Métodos Especiais (`__init__`, `__str__`)
Propriedades (`@property`)
Listas e iteração
Separação de responsabilidades em múltiplos arquivos

>  Observação: a pasta `__pycache__` é criada automaticamente pelo Python e não deve ser incluída no repositório.
